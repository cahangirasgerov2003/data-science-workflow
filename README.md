## 💻 How to Use

### 1️⃣ Clone the repository

```bash
git clone https://github.com/cahangirasgerov2003/data-science-practice.git
cd data-science-practice
```

### 2️⃣ Create a virtual environment (recommended)

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Mac / Linux**

```bash
source venv/bin/activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Jupyter Notebook

```bash
jupyter notebook
```

Open any `.ipynb` file and start exploring the exercises.
